Font weight and italics

The class names for this are prett much self explanatory so not much explanation is provided.

  • sd-text__normal
    // Sets the font style to normal

  • sd-text__strong
    // Sets the font style to bold

  • sd-text__light
    // Sets the font style to light

  • sd-text__italic
    // Sets the font style to italic

  • sd-text__right
    // Aligns the text to the right

  • sd-text__center
    // Aligns the text to the center

  • sd-text__serif
    // Changes the font family to a serif version. Currently set to Merriweather in Superdesk, with a fallback to Georgia, 'Times New Roman', Times, serif;
